Real Time Clock

In this project, I created an analog clock that displays the current time in Pacific Standard Time (PST). The clock dynamically retrieves the real-time data from the browser and converts it into degree format, which is then displayed on the clock's hands.

See the Pen Real Time Clock by Paul Schumacher (@PaulSchumacher) on CodePen.

Random Password Generator

This application allows the user to generate a 14-digit-long random password by pressing the 'Generate' button. The password consists of a combination of lowercase and uppercase letters of the English alphabet, along with numbers and symbols.

See the Pen Untitled by Paul Schumacher (@PaulSchumacher) on CodePen.

Upon pressing on the 'Copy' symbol located to the right of the input box, an alert appears confirming that the password has been copied to the user's computer.

To Do List

This program allows the user to create a personalized to-do list. As a task is entered in the input box, it is automatically added to the list without refreshing the page.

See the Pen To-Do List by Paul Schumacher (@PaulSchumacher) on CodePen.

Once a task is fulfilled, the user can click the green check button to mark the task as done. If the user wants to remove a task, he or she can press the red trash button to delete it from the list.
